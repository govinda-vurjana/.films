# Film Portfolio Website

A modern, responsive film portfolio website with CMS functionality, designed for GitHub Pages deployment.

## Features

- **Hero Section**: Profile image with personal details (height, languages, location, experience)
- **Pinterest-style Gallery**: Masonry grid layout for showcasing your work
- **Admin Panel**: Local CMS for managing images and content
- **GitHub Pages Ready**: Automated deployment workflow
- **Responsive Design**: Works perfectly on all devices
- **Modern Animations**: Smooth transitions and hover effects

## Content Management System

### Admin Panel (`admin.html`)
- **Local Image Management**: Upload, preview, and organize images
- **Drag & Drop**: Easy image uploading interface
- **Gallery Generation**: Creates `gallery.json` for your website
- **Git Integration**: Provides deployment commands

### Workflow
1. Open `admin.html` locally
2. Upload your images using the admin panel
3. Click "Update Gallery" to generate `gallery.json`
4. Save uploaded images to the `assets/` folder
5. Run deployment script to push to GitHub Pages

## GitHub Pages Setup

1. **Enable GitHub Pages**:
   - Go to repository Settings > Pages
   - Source: Deploy from branch `main`
   - Folder: `/ (root)`

2. **Custom Domain** (optional):
   - Add `CNAME` file with your domain
   - Configure DNS settings

## Customization

### Personal Information
Edit the hero section in `index.html`:
- Name and title
- Height, languages, location
- Skills and experience
- Contact details

### Styling
Modify `styles.css`:
- Colors and fonts
- Layout and spacing
- Animations and effects

### Content
- Add images via admin panel
- Update about section
- Modify contact information

## Deployment

### Automatic (Recommended)
```bash
# Windows
deploy.bat

# Mac/Linux  
./deploy.sh
```

### Manual
```bash
git add .
git commit -m "Update portfolio"
git push origin main
```

## Security Notes

- Admin panel (`admin.html`) is for local use only
- Don't deploy admin files to production
- Images are managed locally and committed to Git
- No server-side processing required

## Support

This portfolio is designed for GitHub Pages static hosting. All image management happens locally through the admin panel, ensuring security while providing easy content management.
